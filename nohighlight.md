# `nohighlight`

Indicates that a code block should not be syntax highlighted. 

## Example

This block will not be syntax highlighted:
```HTML
<pre class=nohighlight>
function foo(){
  const a = "foo!";
}
</pre> 
```

But this one will be syntax-highlighted by default:
```HTML
<pre>
function foo(){
  const a = "foo!";
}
</pre> 
```
